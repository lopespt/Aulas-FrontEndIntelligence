Vue.js

Configurações

Crie o arquivo webpack.config.js


var path = require("path");
const VueLoaderPlugin = require('vue-loader/lib/plugin')

module.exports = {
    mode: "development",
    entry: "./index.js",
    output:{
        path: path.resolve(__dirname, "dist"),
        filename: "compilado.js"
    },
    module:{
        rules:[
            {
                test: /\.vue$/,
                use:[
                    'vue-loader'
                ]
            },
            {
                test: /\.css$/,
                use:[
                    'vue-style-loader',
                    "css-loader"
                ]
            }
        ]

    },

    plugins:[
        new VueLoaderPlugin()
    ]
}

Vue.js

Cada componente do vue tem 3 partes:

  • template
  • script
  • style



    <template>
        <h1>Texto do Componente</h1>
    </template>

    <script>
    export default {

    }
    </script>

    <style>

    </style>

Vue.js

Dentro do script, você pode:
  • Definir métodos
  • Dados
  • Propriedades
  • Importar outros componentes e declará-los

Vue.js

Dentro do template, você pode:
  • Escrever HTML
  • Utilizar propriedades e dados do componente através da tag mustache "{{ }}"
  • Iterar sobre vetores
  • Associar classes e estilos
  • Disparar eventos

Vue.js

Dentro do style, você pode:
  • Definir classes e regras de CSS
  • Optar por essas regras estarem ou não somente no escopo do componente
  • Associar classes e estilos
