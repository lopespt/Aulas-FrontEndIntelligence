Jekyll - Parte 2

Criando seu primeiro site Jekyll

  • A primeira coisa é baixar e instalar o Ruby
  • Abra um terminal do windows com "CTRL+r" e digite "cmd".
  • Agora vamos instalar o pacote jekyll usando o ruby gem (gerenciador de pacotes)
    • 
$ gem install jekyll bundler
  • Caso queira maiores informações para a instalação, acesse aqui

Linha de comando do Jekyll

  • jekyll new <nome> :cria um novo site na pasta <nome>
  • jekyll new --blank <nome> : cria um site jekyll em branco
  • jekyll build Faz a compilação do site na pasta "./_site/"
  • jekyll serve Compila o site e cria um servidor local (para testes)
  • jekyll doctor retorna problemas com a configuração do seu jekyll
  • jekyll clean limpa arquivos temporários e a pasta "./_site/"
  • jekyll help Ajuda do sistema

Seu primeiro Site

  • Entre no terminal do windows
  • Vá para algum diretório de sua escolha usando "cd"
  • digite: 
    $ jekyll new jekyll_inicio
  • Nesse instante uma nova pasta será criada com um template padrão

Visualizando seu Site

  • Para conseguir visualizar seu site, você deverá rodar de dentro da pasta do projeto: 
    $ jekyll build
  • E, depois: 
    $ jekyll serve
  • Prontinho! Verifique como está seu site em: http://localhost:4000

Conteúdos no Jekyll

  • Há diversas maneiras de se adicionar conteúdo a um site Jekyll.
    • Pela pasta _posts: Podemos criar um arquivo na pasta obedecendo o padrão de nomeação. O jekyll irá adicionar um novo tópico ao site.
    • Novo arquivo em qualquer pasta: veremos mais tarde
    • Uso de dados: veremos mais tarde
    • Uso de coleções: veremos mais tarde

Criação de um post

  • Para adicionar um novo post, basta criar um arquivo na pasta _posts/ com o seguinte padrão de nome:
  • ANO-MES-DIA-titulo.tipo
  • Exemplo: 2019-02-12-Aula-de-jekyll.md

Front Matter

  • Tratam-se das informações que precedem o conteúdo de um arquivo Jekyll.
  • São escritas com YAML entre ---
  • Exemplo: 
    ---
title: "Meu título"
date: 2019-02-10 23:28:37 -0200
---
Esse é o meu conteúdo...blablabla

Agora é a sua vez

  • Crie um novo post no blog padrão usando html e outro usando markdown
  • O front matter dos seus posts deverá seguir o padrão: 
    ---
title: <Escolha qualquer um>
date: 2019-02-10 23:28:37 -0200 <pode mudar, desde que obedeça o padrão>
---
Esse é o meu conteúdo...blablabla

Atualizando seu site

  • Para conseguir visualizar suas alterações e novos posts, você deverá rodar: 
    $ jekyll build
  • E, depois: 
    $ jekyll serve
  • Prontinho! Verifique como está seu site em: http://localhost:4000

Criando um layout

  • Para a criação de um novo layout, devemos adicionar um novo arquivo na pasta _layouts
  • Esse arquivo só poderá ser escrito em HTML
