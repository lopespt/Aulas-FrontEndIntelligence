Jekyll

Jekyll

  • É um framework para criação de páginas estáticas criado através da linguagem Ruby.
  • Integra YAML, JSON, Markdown e Liquid template parser em um só ambiente
  • Seguro e prático!
  • O GitHub tem suporte para publicar através de Jekyll!
  • Não é possível criar páginas dinâmicas do lado do servidor

Jekyll

Quick Start!

Documentação 

~: gem install jekyll bundler
O código acima irá instalar os pacotes GEM (ruby) em sua máquina. Esse processo é semelhante a um 
npm install -g

Jekyll

Quick Start!

Para criar um site padrão: 

~: jekyll new <nome do site>
Para iniciar um servidor local: (obs: você precisa estar dentro da pasta do projeto) 

~: jekyll serve

Jekyll

Liquid

  • Liquid é uma linguagem para criação de templates/padrões de páginas.
  • O Jekyll lê todos os códigos (Markdown, HTML, JS, CSS, YAML, JSON, Liquid) e compila para o diretório _site.

Jekyll

Liquid

  • Há 3 partes importantes no Liquid:
    • Objetos 
      
{{ page.title }}
    • Tags 
      
{% if page.show_sidebar %}
<div class="sidebar">
    sidebar content
</div>
{% endif %}
    • Filtros 
      
{{ "hi" | capitalize }}

Jekyll

Front Matter

  • É um cabeçalho da sua página
  • Aqui se define em YAML ou JSON as propriedades que serão utilizadas pelo Jekyll
  • Deve-se sempre ser colocada no início de arquivo entre --- 
    
---
minha_variavel: 29
---
  • Essa variável poderá ser utilizada na página: 
    
{{ page.my_number }}

Jekyll

Exemplo de uma página


---
title: Home
---
<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <title>{{ page.title }}</title>
  </head>
  <body>
    <h1>{{ "Hello World!" | downcase }}</h1>
  </body>
</html>
